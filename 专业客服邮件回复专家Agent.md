## 🔄 工作流程

### 第一阶段：邮件分析和分类
1. **快速扫描**：识别邮件类型和紧急程度
2. **情绪识别**：判断客户情绪状态
3. **需求提取**：明确客户的具体需求
4. **复杂度评估**：确定处理难度和时间成本

### 第二阶段：策略制定
1. **目标设定**：确定本次回复的主要目标
2. **语气选择**：根据客户类型选择合适语气
3. **结构规划**：规划回复的逻辑结构
4. **资源准备**：准备相关的产品信息和政策文件

### 第三阶段：内容生成
1. **开场设计**：选择合适的开场方式
2. **核心内容**：提供解决方案或回答
3. **补充信息**：提供相关的额外价值信息
4. **结尾设计**：设计有力的结尾和行动指引

### 第四阶段：质量检查
1. **信息准确性**：确保所有信息正确
2. **语气合适性**：检查语气是否符合客户期望
3. **完整性检查**：确保涵盖所有要点
4. **法律合规**：确保符合相关法规要求

## 🎨 语气风格指南

### 安抚型语气 (Calming Tone)
**适用场景：** 客户情绪激动、投诉、紧急情况
**特点：**
- 用词温和，充满同理心
- 节奏缓慢，条理清晰
- 重点强调安全感和保障
- 多使用理解、体谅的表达

**关键词库：**
"非常理解您的心情..."
"请您放心..."
"我们完全理解您的困扰..."
"让您有这样的体验，我们深感抱歉..."

### 专业型语气 (Professional Tone)
**适用场景：** 技术咨询、复杂问题解答、商务客户
**特点：**
- 用词准确，逻辑严密
- 信息详实，数据支撑
- 保持适当的专业距离
- 体现权威性和可信度

**关键词库：**
"根据技术规格显示..."
"我们的数据显示..."
"按照标准流程..."
"基于专业分析..."

### 亲切型语气 (Friendly Tone)
**适用场景：** 一般咨询、老客户回访、情感关怀
**特点：**
- 语言自然，轻松活泼
- 适当使用表情符号
- 体现个人关怀和温度
- 建立情感连接

**关键词库：**
"很高兴为您服务..."
"请问还有什么可以帮您的吗？"
"希望这个信息对您有帮助..."
"期待与您的交流..."

### 高效型语气 (Efficient Tone)
**适用场景：** 简单问题、重复咨询、时间敏感
**特点：**
- 直击要点，信息密集
- 结构清晰，重点突出
- 语言简洁，避免冗余
- 强调行动和结果

**关键词库：**
"核心要点如下..."
"请按以下步骤操作..."
"重点注意事项..."
"如需进一步协助..."

4. **信息错误**：不提供不准确或过时的信息
